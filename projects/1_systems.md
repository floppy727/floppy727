# Systems & Low-Level

Projects focused on kernel development, memory management, embedded systems and low-level programming.

---

**1000Hz Tablet — Custom Pen Tablet Firmware (Embedded/C/C++) (PROTOTYPING PHASE)**  
Reverse-engineering a Wacom CTH480 EMR tablet to amplify from 133Hz to 1000Hz. Coil signals via ADC, reconstructing pen position, streaming high-rate USB HID reports.  
`C/C++ · ESP32 · ADC · Performance · Signal Processing · USB HID · Embedded · Reverse Engineering`  
[![Repo](https://img.shields.io/badge/GitHub-1000hz__tablet-181717?logo=github)](https://github.com/floppy727/1000hz_tablet)

**KFS — Kernel From Scratch (ASM) (WIP 3/10)**  
Series of 10 projects to build a full UNIX kernel from scratch mainly in Assembly. IDT/GDT, memory & process management, syscalls, filesystem, modules, full environment.  
`Kernel · Assembly · OS internals · Memory · Low-level · Systems`  
[![Repo](https://img.shields.io/badge/GitHub-KFS-181717?logo=github)](https://github.com/floppy727/KFS)

**minishell — Unix Shell (C)**  
Bash-like shell: parsing, pipes, redirections, signals, process control.  
`C · Low-level · Lexing · Parsing · Unix · Memory`  
[![Repo](https://img.shields.io/badge/GitHub-42__minishell-181717?logo=github)](https://github.com/floppy727/42_minishell)

**ft_malloc — Custom Memory Allocator (C)**  
Implementation of a custom `malloc`, `free`, and `realloc` — managing memory zones, page allocation and fragmentation.  
`C · Memory · Low-level · Systems · Unix`  
[![Repo](https://img.shields.io/badge/GitHub-ft__malloc-181717?logo=github)](https://github.com/floppy727/ft_malloc)

**ComputorV1 — Polynomial Equation Solver (Rust)**  
Parser and solver for polynomial equations up to degree 2. Tokenization, reduction, discriminant computation, and formatted step-by-step resolution.  
`Rust · Parsing · Algorithms · CLI · Mathematics · Precision`  
[![Repo](https://img.shields.io/badge/GitHub-ComputorV1-181717?logo=github)](https://github.com/floppy727/ComputorV1)

**libasm — Assembly Library (ASM)**  
Reimplementation of standard library functions in x86-64 Assembly.  
`Assembly · Low-level · Systems`  
[![Repo](https://img.shields.io/badge/GitHub-libasm-181717?logo=github)](https://github.com/floppy727/libasm)
